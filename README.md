# 35 PERI Systems Demo

PERI systems project management tool demo.

## Setup
Clone the project

To use the application, you will need to run both servers. 

### Node Server
1. Ensure you're in the directory `./Backend`
2. `npm install`
3. `npm run dev`

### React Server
1. Ensure you're in the directory `./Frontend`
2. `npm install`
3. `npm run start`
